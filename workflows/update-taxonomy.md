# Update Taxonomy Workflow

1. Identify whether the task changes keywords, themes, or theme links.
2. Check for existing near-duplicate themes before creating a new one.
3. Edit the canonical metadata or theme object with the smallest necessary change.
4. Re-check any paper linked to a renamed or split theme.
5. Sync canonical markdown back into SQLite.
6. Re-export website data when the paper is publicly displayed.
