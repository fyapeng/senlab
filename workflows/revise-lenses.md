# Revise Lenses Workflow

1. Read the current paper card plus existing excerpts and lenses.
2. Identify whether the request affects excerpts, lenses, or both.
3. Preserve IDs when the analytic function stays the same.
4. Add or replace only the necessary evidence blocks.
5. Tighten overclaim risk and safer formulation when user supplements narrow the claim.
6. Sync canonical markdown back into SQLite.
7. Re-export website data when the paper is publicly displayed.
